# 🧮 Calculadora de IMC

Projeto desenvolvido para praticar os fundamentos de **HTML, CSS e JavaScript**, criando uma aplicação web funcional para cálculo do Índice de Massa Corporal (IMC).

A aplicação permite que o usuário informe seu peso e altura, realize o cálculo automaticamente e visualize sua classificação conforme as faixas de IMC.

## 📌 Funcionalidades

✅ Cálculo automático do IMC  
✅ Validação de campos vazios  
✅ Validação de valores inválidos  
✅ Classificação do resultado:
- Abaixo do peso
- Peso normal
- Sobrepeso
- Obesidade Grau I
- Obesidade Grau II
- Obesidade Grau III

✅ Layout responsivo para diferentes tamanhos de tela

---

## 🧠 Conceitos praticados

Durante o desenvolvimento deste projeto foram aplicados:

- Estruturação semântica com HTML5
- Estilização utilizando CSS3
- Flexbox para organização dos elementos
- Responsividade com Media Queries
- Manipulação do DOM com JavaScript
- Eventos com `addEventListener`
- Validação de formulários
- Conversão de valores utilizando `Number()`
- Uso de condicionais (`if`, `else if`, `else`)
- Controle de fluxo com `return`
